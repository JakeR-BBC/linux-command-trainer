# Linux Command Trainer

A browser-based tool for learning and drilling Linux commands — no terminal access required.

## Features
- Three drill modes: Recognition, Recall, and Scenario
- Six command categories: Navigation, Files, Permissions, Search, Processes, Networking
- Command retirement system to focus on weak spots
- Progress tracked locally in the browser

## Getting Started

### Prerequisites
- Node.js v22+ (use `nvm use` if you have nvm installed)

### Install and run
```bash
npm install
npm run dev
```
